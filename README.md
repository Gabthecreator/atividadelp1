Repositorio das questões de LP1

Vai precisar de um compilador C GCC/MinGW, ou o CLion instalado que foi o que usei

### Para executar via terminal
 Para todas as 6 questões seguindo os passos trocando apenas o "1" pelo numero da questao desejada
 
**Compilar:**
`gcc questao1.c -o questao1`

**Executar (Windows):**
`.\questao1.exe`

**Executar (Linux/Mac):**
`./questao1`

## Executar via IDE (CLion / CMake)

Fazer clone desse repositório no seu terminal

Abrir o CLion, ir em `File` > `Open` e selecione a pasta raiz do projeto.

Após a IDE configurar o ambiente de compilção basta rodar
